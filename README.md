
# Cypress Automation Test - Microsoft 365 Office

### 🛠️ Tech Stack
- [Cypress](https://www.cypress.io/)

### ⚙️ Installation

1. Clone the repository:
   ```bash
   cd your-repo-name
   git clone https://github.com/novalrizkyangkasa/technical_test_lnk.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### 🚀 How to Run Tests
